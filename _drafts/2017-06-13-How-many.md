## 媒体查询

媒体查询是为了针对移动端的Web网页。

1. 在网页的`head`加入下面的代码(`<title>`上面是个不错的选择)

```html
<meta name="viewport" content="width=device-width, inital-scale=1.0">
```

2. 在`CSS`中引入媒体查询

```css
@media (min-width: 481px) and (max-width:768px){
    /*样式在这里*/
}
```
