## 1. 没有负效应的 Hash 表/图

```js
const map = Object.create(null);
```

如果采用字面值形式 `const map = {}`，等同于 `Object.create(Object.prototype)`。从而，map 会继承来自 Object 的一些熟悉和方法。使用上述方法创建的 hash 表/图 什么都不会从 Object 继承。
